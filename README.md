# 조작방법

- 키보드
    - **이동(2개)** : ad
    - **스킬(5개)** : space, u, i, o, p
- 터치 스크린
- Xbox 컨트롤러 :
    - **이동(2개)** : leftStick
    - **스킬(5개)** : X, Y, A, B, RightButton

# 개발스텍

- Unity 6.2.14f1 
- VisualStudio 2022
- UniTask.
- Ugui, Ui-Extension
- InputSystem, Timeline
- Urp-2D, Sprite, Sprite-Atlas, 2D-Skinning, Volume(Bloom)
- Physics2D Rigidbody2D, Collider2D
